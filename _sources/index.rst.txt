***********************************************
SagemakerExperimentsLogger  Documentation
***********************************************

SagemakerExperimentsLogger provides a simple way to log experimental data such as hyperparameter settings and 
evaluation metrics via AWS SageMaker Experiments API. It can be easily integration into the concept of Pytorch 
Lightning Trainer class.
